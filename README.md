# -
Модель клеточного автомата «Игра Жизнь» Конвея с расширениями
1.	бесконечное поле;
2.	эффективное хранение клеток;
3.	выбор различных правил клеточных автоматов;
4.	загрузка и установка паттернов;
5.	возможность шага назад;
6.	управление скоростью симуляции;
7.	перемещение поля.

Клеточные автоматы
Клеточный автомат представляет собой:
1.	решётку клеток
2.	набор состояний клеток
3.	правила перехода между состояниями
4.	дискретные шаги времени
Каждая клетка изменяет своё состояние на следующем шаге на основе локальных правил, которые учитывают состояние соседних клеток.
Игра «Жизнь» Конвея
Поле состоит из клеток, каждая из которых может быть:
1.	живой
2.	мёртвой
Для каждой клетки рассматриваются 8 соседей.





Основные правила (B3/S23)
Обозначение B/S означает:
•	B (Birth) - рождение
•	S (Survive) - выживание
Правила:
1.	Мёртвая клетка оживает, если у неё 3 соседа.
2.	Живая клетка выживает, если у неё 2 или 3 соседа.
3.	В остальных случаях клетка умирает.
Клавиши:
W	вверх
A	влево
S	вниз
D	вправо
